syncBackup relies on third party applications that must be previously installed in your system.

The expected executables should be in any of the standard locations for binaries registered in PATH.

RSYNC

Back-end required for pretty much all the functionality in this program.

Executable: rsync

SSH-ASKPASS

Required when the source or destination is using the shell to connect to a remote host via SSH. This application handles the authentication process when a password is requested from the user.

Executable: ssh-askpass

Note: Flatpak package includes all dependencies.
